# Console Catalog API

Node.js REST API for iconic consoles, brands, games, franchises, users, and reviews.

## Project Idea

This API stores console catalog history in linked collections:

- Brands, such as Nintendo, Sega, Microsoft, Atari, and Sony
- Consoles, such as NES, GameCube, Xbox 360, Game Boy Advance, and Nintendo 3DS
- Franchises, such as Mario, Zelda, Halo, Sonic, and Pokemon
- Games, linked to consoles, brands, and franchises
- Users, used for authentication and review ownership
- Reviews, linked to a user and a game
